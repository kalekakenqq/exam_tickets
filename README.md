# Экзаменационные билеты — Инструкция по запуску

## Требования
- Python 3.10+ установлен и добавлен в PATH
- Интернет-соединение для билетов 11 (скачивание картинок) и 13 (проверка сайтов)

---

## Flask (билеты 01–05)

**01** — визитка, **02** — конвертер температур, **03** — викторина, **04** — REST API, **05** — корзина

### Запуск

1. Открыть папку билета в VSCode / PyCharm (`File → Open Folder`)
2. Открыть терминал внутри папки
3. Выполнить:

```
pip install flask
python app.py
```

4. Открыть в браузере: **http://127.0.0.1:5000**
5. Остановить сервер: `Ctrl+C`

> **Билет 04** (REST API) — браузер покажет пустую страницу на `/`, это нормально.
> Данные доступны по адресу `http://127.0.0.1:5000/api/notes`

---

## Django (билеты 06–10)

**06** — блог, **07** — список задач, **08** — авторизация, **09** — форма обратной связи, **10** — фильмы

### Запуск

1. Открыть папку билета в VSCode / PyCharm
2. Открыть терминал внутри папки
3. Выполнить:

```
pip install django
python manage.py migrate
python manage.py runserver
```

4. Открыть в браузере: **http://127.0.0.1:8000**
5. Остановить сервер: `Ctrl+C`

> **Билеты 06 (блог) и 10 (фильмы)** — список будет пустым после первого запуска.
> Чтобы добавить данные, создать суперпользователя и войти в админку:
>
> ```
> python manage.py createsuperuser
> ```
>
> Затем открыть **http://127.0.0.1:8000/admin** и добавить посты / фильмы.

> **Билеты 07, 08, 09** — данные добавляются прямо через интерфейс сайта.

## SQLite (билеты 16–20)

**16** — студенты, **17** — авторизация, **18** — транзакции, **19** — библиотека, **20** — индексы

### Запуск

1. Открыть папку билета в VSCode / PyCharm
2. Открыть терминал внутри папки
3. Выполнить:

```
python solution.py
```

Никаких библиотек устанавливать не нужно.

> **Билет 17** (авторизация) — интерактивный: в терминале появится меню `1. регистрация  2. вход  3. выход`.

## Потоки / Очередь (билеты 11, 12, 15)

**11** — параллельное скачивание картинок, **12** — банкомат (race condition), **15** — producer/consumer

### Запуск

```
python solution.py
```

Никаких библиотек устанавливать не нужно.

> **Билет 11** скачивает изображения с picsum.photos — нужен интернет.
> Изображения сохраняются в папку `downloads/` внутри папки билета.

## Async (билеты 13, 14)

**13** — сканер сайтов, **14** — таймер с паузой/остановом

### Запуск

```
pip install aiohttp nest_asyncio
python solution.py
```

> **Билет 13** — нужен интернет (проверяет google.com, github.com и др.).
>
> **Билет 14** — после запуска ввести количество секунд, затем:
> - `p` + Enter — пауза / продолжить
> - `q` + Enter — выйти досрочно
